Viewport wrapper

Width & height al 100% del contenedor padre con display flex. No se hace uso de unidades como vw o vh, ya que estas dan problemas en algunos dispositivos moviles. En su lugar se recomienda el uso del reset.css de reset utilities.

Ejemplo:

.viewport

Scroll wrapper

Como la clase .viewport pero con el overflow-y en auto. Se recomienda el uso de esta clase en todo lugar donde se necesite scroll vertical, ya que incluye correcciones para este proposito, asi como dise√±o personaliable en la barra.

Ejemplo:

.scroll

Snap scroll vertical

Al usar la clase .snap sobre contenedores con la clase .scroll, estos se comportan con scroll snap en el eje vertical. La primera generacion de hijos actuaran como secciones del scroll snap.

Ejemplo:

.scroll.snap (view 1)
.scroll.snap (view 2)
.scroll.snap (view 3)

Slider & slide wrappers

Clases para generar scroll horizontal. La clase .slide siempre tiene que ser un hijo directo de la clase .slider. Para asegurar el correcto funcionamiento, se recomienda usar un hijo dentro de la clase .slide y a este aplicarle los estilos deseados. Si se usa la clase .snap en .slider, este obtendra snap scroll horizontal y usara los .slide como secciones.

Ejemplo: (shift + scroll para dispositivos no touch)

.slider.snap > .slide (view 1)
.slider.snap > .slide (view 2)
.slider.snap > .slide (view 3)

Row direction

Clase que define la direccion del contenedor como una fila. Aplica display flex con wrap.

Ejemplo:

.row > element 1
.row > element 2
.row > element 3
